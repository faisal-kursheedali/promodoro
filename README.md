

## Promodoro


![pomodoro-logo](https://user-images.githubusercontent.com/85350108/201940860-bdf6aa9f-da11-4221-9d59-b127356f4953.png)



<h3>Promodoro</h3> 
<div>Promodoro is a fully functioning dynamic web application. It is a Promodoro web application that is built using HTML, CSS, TypeScript and React-router. It has all the basic features of a Promodoro application.</div>

<h4>Pages</h4>
<ul>
  <li>Home</li>
  <li>Completed / Completed task listing page</li>
  <li>Tag / Tasks of same tags are listed here</li>
  <li>Timer</li>
</ul>
<h4>Features</h4>
<ul>
  <ul>
<li>Add a task</li>
<li>Edit/Delete a task</li>
<li>Pomodoro Feature</li>
<li>Start/Pause/Reset Clock</li>
<li>Data Persistence</li>
<li>Tags for task</li>
<li>Page title in sync with Pomodoro time</li>
</ul>
  
  
  
</ul>
